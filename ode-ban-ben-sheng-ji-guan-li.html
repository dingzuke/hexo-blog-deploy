升级或降级node,网上找了一大堆的资料,看着头皮发麻,只需两行代码的事情
    1. sudo npm install -g n  // 安装
    2. sudo n v9.10.0  // 升级到某个版本  如:v9.10.0
    3. sudo n stable   // 升级到最稳当版本
